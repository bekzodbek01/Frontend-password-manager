# Frontend-password-manager

 # 🌐 Frontend (React)

Bu loyiha frontend qismi React yordamida yozilgan.

---

## Clone

git clone https://github.com/bekzodbek01/password-manager.git  
cd password-manager/frontend

---

## Install

npm install

---

## Run

npm start

👉 http://localhost:3000

---

## Muhim

Backend ishlashi kerak:
http://127.0.0.1:8000

---

## Ishlatish

1. Register → username, password, master password  
2. Login → username + password  
3. Unlock → master password  
4. Add / Edit / Delete → credential boshqarish  

---

## Eslatma

- Master passwordni unutmang  
- Token localStorage’da saqlanadi
